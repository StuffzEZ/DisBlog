To customise the template, please make sure you dont remove any of the existing elements and don't do the following things:

  • Remove the <script> tag - this is for comments
  • Don't change/remove the things in curly brackets (eg. {{ discussion.id }} )

Tags

{{ discussion.id }} - The discussion ID
{{ discussion.title }} - The discussion Title
{{ discussion.content }} - The discussion content
{{ repo.path }} - The repository path
{{ blog.name }} - The name of the blog (set as a variable)
